# 블로그 콘텐츠 작성 가이드

본 프로젝트는 플랫폼별(네이버, 티스토리, 애드센스, 브랜드) 블로그 글 작성 규칙과 템플릿을 제공합니다.

## 폴더 구조

```
blog/
├── guidelines/          # 플랫폼별 작성 가이드라인
├── templates/          # 플랫폼별 템플릿
└── examples/           # 실제 작성 예제
```

## 사용 방법

1. `guidelines/` 폴더에서 해당 플랫폼의 작성 규칙 확인
2. `templates/` 폴더에서 템플릿 복사
3. `examples/` 폴더에서 실제 예제 참고
