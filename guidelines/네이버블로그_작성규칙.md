# 네이버 블로그 작성 규칙

## 핵심 원칙: PLAIN TEXT 전용

네이버 블로그는 **마크다운을 절대 사용하지 않습니다**.

## 금지 사항

### 절대 사용 금지 기호
- `#` (제목 표시)
- `##` (소제목 표시)
- `**` (볼드체)
- `*` (이탤릭)
- `-` (리스트)
- `|` (테이블)
- `---` (구분선)
- ``` (코드 블록)
- 기타 모든 마크다운 문법

## 필수 출력 포맷

```
[제목]
(여기에 제목 작성)

[본문]
(여기에 본문 작성 - 순수 텍스트만)

[마무리]
(핵심 요약 3-5줄)

[태그]
#태그1 #태그2 #태그3 #태그4 #태그5

[썸네일 아이디어]
1. (구체적인 썸네일 아이디어)
2. (구체적인 썸네일 아이디어)
```

## 콘텐츠 작성 원칙

### 1. 분량
- 기본: 2,500~3,500자
- 사용자 지정 시 해당 분량 준수

### 2. 심층 작성 (충실한 분량)
- 각 주제/섹션당 **반드시** 최소 6-8문장 이상 작성
- 피상적인 설명 배제, 심층적인 내용으로 구성
- 구체적인 수치, 통계, 실제 사례 필수 포함
- 실용적인 팁과 데이터 기반 정보 제공
- 추상적 표현 지양, 구체적이고 실질적인 설명 필수

### 3. 팩트 기반 작성 (허위 경험 절대 금지)
- 객관적 정보와 검증된 사실만 활용
- **절대 금지**: "제가 직접 써보니", "어제 다녀온", "직접 사용해보니", "제가 먹어본 결과" 등 가짜 체험담
- 허위 사용 후기 및 경험담 작성 절대 금지
- 출처가 명확한 데이터와 통계만 활용
- 구체적인 수치, 실제 사례, 실용적인 팁 포함하여 신뢰도 제고

### 4. 후킹 도입부
- 흥미로운 수치나 통계로 시작
- 독자의 궁금증을 유발하는 질문 활용
- 글을 읽어야 할 명확한 이유 제시
- 예시: "2024년 기준 70% 이상이 이 방법을 모르고 있습니다"

### 5. 가독성
- 한 문단: 3-5문장 구성
- 적절한 줄바꿈으로 시각적 여백 확보
- 문장은 간결하고 명확하게
- 전문 용어 사용 시 쉬운 설명 병기

### 6. 구조화
- 논리적 흐름: 도입 → 전개 → 심화 → 마무리
- 자연스러운 주제 전환
- 섹션 구분은 줄바꿈과 내용으로만 표현 (기호 사용 금지)

### 7. 필수 포함 요소
- **핵심 요약(마무리)**: 본문의 주요 내용을 3-5줄로 압축
- **썸네일 아이디어**: 구체적이고 실현 가능한 아이디어 2개
- **태그**: 5-10개 (# 기호 사용)

## 작성 체크리스트

- [ ] 마크다운 기호 전혀 사용하지 않았는가?
- [ ] [제목], [본문], [마무리], [태그], [썸네일 아이디어] 포맷을 준수했는가?
- [ ] 각 섹션이 6-8문장 이상인가?
- [ ] 가짜 체험담이나 허위 정보가 없는가?
- [ ] 후킹 도입부가 있는가?
- [ ] 문단별 3-5문장 구성이 지켜졌는가?
- [ ] 핵심 요약과 썸네일 아이디어가 포함되었는가?
- [ ] 태그가 # 형식으로 작성되었는가?
- [ ] 분량이 2,500~3,500자인가?

## 주의사항

1. **절대 포함하지 말 것**
   - "이미지를 삽입하세요" 같은 메타 가이드
   - 의미 없는 문장 반복으로 분량 채우기 (늘어쓰기 금지)
   - SEO 키워드 나열식 작성
   - 가짜 경험담이나 허위 체험 정보

2. **품질 우선**
   - 분량 채우기보다 내용의 깊이
   - 실질적 가치 제공
   - 독자 중심 관점
