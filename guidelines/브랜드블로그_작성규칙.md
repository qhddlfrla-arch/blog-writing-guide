# 브랜드 블로그 작성 규칙

## 핵심 원칙: 마크다운 필수 + 브랜드 정체성

브랜드 블로그는 **마크다운을 적극 활용**하며 **브랜드 가치와 전문성**을 전달합니다.

## 필수 사용 마크다운

### 1. 제목 구조
```markdown
# 메인 제목 (브랜드 관점 명확히)

## 주요 섹션 제목

### 소제목 (필요시에만 최소한으로 사용)
```

### 2. 강조
```markdown
**핵심 메시지 강조**
*보조 강조*
```

### 3. 리스트
```markdown
- 고객 가치 항목
- 서비스/제품 특징
  - 상세 설명

1. 단계별 가이드
2. 프로세스 설명
```

### 4. 테이블 (비교, 가격 정보 등)
```markdown
| 항목 | 내용 | 특징 |
|------|------|------|
| A | 설명A | 특징A |
| B | 설명B | 특징B |
```

### 5. 인용구 (고객 후기, 전문가 의견)
```markdown
> "실제 고객 후기 또는 전문가 의견"
> - 출처 명시
```

## 출력 포맷

```markdown
# 메인 제목 (브랜드 메시지 포함)

후킹 도입부 (고객의 pain point 또는 니즈 언급)

## 배경/문제 인식

고객이 직면한 문제나 시장 상황 설명 (6-8문장)

## 솔루션/제품 소개

브랜드의 해결책 제시 (6-8문장)
- 핵심 특징
- 차별화 포인트
- 고객 가치

## 상세 정보

### 필요시 하위 섹션

구체적인 정보, 데이터, 사례

| 비교 항목 | 내용 |
|-----------|------|
| 항목1 | 설명1 |
| 항목2 | 설명2 |

## 실제 사례/고객 사례

> "실제 고객 후기"
> - 고객명 (선택)

구체적 성과나 결과

## 마무리

핵심 메시지 요약 및 Call-to-Action

---

**태그**: 태그1, 태그2, 태그3, 태그4, 태그5

**썸네일 아이디어**:
1. 브랜드 아이덴티티를 반영한 아이디어
2. 고객 관심을 끄는 구체적 아이디어
```

## 콘텐츠 작성 원칙

### 1. 분량
- 기본: 2,500~3,500자
- 심층 콘텐츠: 3,500자 이상
- 사용자 지정 시 해당 분량 준수

### 2. 브랜드 톤앤매너
- 브랜드 정체성에 맞는 문체
- 전문적이면서도 친근한 어조
- 일관된 용어 사용
- 브랜드 가치 자연스럽게 전달

### 3. 제목 사용 규칙 (## 중심 구조)
- `#` 메인 제목: 1회만, 브랜드 메시지 포함
- `##` 주요 섹션: 3-5개, 논리적 흐름
- `###` 소제목: **가급적 지양**, 꼭 필요한 경우에만 최소한으로 사용
- ## 중심의 명확한 구조로 브랜드 메시지 효과적 전달

### 4. 심층 작성 (충실한 분량)
- 각 섹션당 **반드시** 최소 6-8문장 이상 작성
- 피상적인 설명 배제, 심층적인 내용으로 구성
- 구체적인 데이터와 사례 필수 포함
- 고객 중심 관점 유지
- 실질적 가치 제공
- 전문성 입증
- 표와 리스트 활용한 정보 구조화

### 5. 팩트 기반 작성 (허위 경험 절대 금지)
- 객관적이고 검증된 정보만 사용
- 과장 금지, 정직한 소통 원칙
- **실제 고객 사례만 사용** (허가받은 내용만)
- **절대 금지**: "제가 직접 써보니", "어제 다녀온", "직접 사용해보니" 등 가짜 체험담
- 허위 후기 작성 절대 금지
- 데이터 출처 반드시 명확히 표기
- 주장에 대한 근거 제시 필수

### 6. 고객 중심 접근
- 고객의 문제 인식에서 시작
- 브랜드 중심이 아닌 고객 중심
- "우리가 최고" 대신 "고객 문제 해결"
- 실질적 혜택과 가치 강조

### 7. 후킹 도입부
- 고객의 pain point 직접 언급
- 공감대 형성
- 명확한 가치 제안
- 흥미로운 데이터나 인사이트

### 8. 가독성
- 한 문단: 3-5문장 구성
- 마크다운으로 구조화
- 표와 리스트 활용
- 시각적 여백 확보
- 전문 용어는 설명 추가

### 9. 필수 포함 요소
- **핵심 요약**: 브랜드 메시지와 고객 가치
- **Call-to-Action**: 자연스러운 다음 단계 제시
- **썸네일 아이디어**: 브랜드 정체성 반영
- **태그**: 쉼표로 구분 (태그1, 태그2)

## 브랜드 블로그 특화 원칙

### 1. 콘텐츠 유형별 접근

#### 제품/서비스 소개
- 고객 문제 → 솔루션 → 특징 → 사례 → CTA
- 차별화 포인트 명확히
- 과장 없는 정직한 설명

#### 업계 인사이트
- 시장 트렌드 분석
- 전문가 관점 제시
- 브랜드 전문성 입증
- 고객에게 유용한 정보

#### 고객 사례/성공 스토리
- 실제 사례만 사용
- 구체적 성과 수치
- 고객 허가 받은 정보만
- 다른 고객에게 참고가 되는 내용

#### 가이드/튜토리얼
- 단계별 명확한 설명
- 실용적 팁 제공
- 브랜드 제품/서비스 자연스럽게 연결
- 독립적 가치 제공

### 2. 브랜드 메시지 통합
- 강요하지 않고 자연스럽게
- 고객 가치와 연결
- 일관된 브랜드 보이스
- 진정성 있는 소통

### 3. 신뢰 구축
- 투명한 정보 제공
- 출처 명확한 데이터
- 객관적 분석
- 고객 프라이버시 존중

### 4. 참여 유도
- 자연스러운 CTA
- 댓글/피드백 유도
- 다음 단계 제시
- 고객과의 대화 촉진

### 5. 금지 사항
- 경쟁사 비방
- 과장 광고
- 허위 정보 및 가짜 경험담
- 강압적 판매
- 가짜 후기/사례 (허가받지 않은 내용)
- 브랜드 자화자찬만 (고객 가치 무시)
- 의미 없는 문장 반복 (늘어쓰기 금지)

### 6. 권장 사항
- 고객 성공에 집중
- 교육적 가치 제공
- 솔직하고 투명한 소통
- 장기적 관계 구축
- 업계 리더십 입증

## 분량 가이드

- **기본**: 2,500~3,500자
- **심층 콘텐츠**: 3,500~4,500자
- **간단한 공지**: 1,500자 이상
- **중요**: 품질 우선, 필요한 만큼만

## 태그 전략

- 브랜드 관련 키워드
- 제품/서비스 카테고리
- 고객 관심사
- 업계 트렌드
- 5-10개 적정
