# 애드센스 블로그 작성 규칙

## 핵심 원칙: 마크다운 필수 + SEO 최적화

애드센스 블로그는 **마크다운을 적극 활용**하며 **검색 최적화**를 고려합니다.

## 필수 사용 마크다운

### 1. 제목 구조 (SEO 중요)
```markdown
# 메인 제목 (키워드 포함, 1회만 사용)

## 주요 섹션 제목 (관련 키워드 자연스럽게)

### 소제목 (필요시에만 최소한으로 사용)
```

### 2. 강조
```markdown
**중요 키워드 강조**
*보조 강조*
```

### 3. 리스트
```markdown
- 핵심 정보 항목
- 구체적 데이터
  - 상세 설명

1. 단계별 설명
2. 순서가 있는 정보
```

### 4. 테이블 (비교 콘텐츠에 효과적)
```markdown
| 항목 | 특징 | 장점 | 단점 |
|------|------|------|------|
| A | 설명A | 장점A | 단점A |
| B | 설명B | 장점B | 단점B |
```

## 출력 포맷

```markdown
# 메인 제목 (키워드 포함)

후킹 도입부 (2-3문장, 키워드 자연스럽게 포함)

## 섹션 1 제목

본문 내용 (6-8문장 이상)
- 구체적 정보
- 실제 데이터
- 검증된 사례

## 섹션 2 제목

본문 내용 (6-8문장 이상)

**핵심 키워드** 자연스럽게 활용

### 필요시 소제목 (최소한으로)

상세 내용

## 섹션 3 제목

| 비교 항목 | 설명 |
|-----------|------|
| 내용1 | 설명1 |
| 내용2 | 설명2 |

## 마무리

핵심 요약 (3-5문단)

---

**태그**: #태그1 #태그2 #태그3 #태그4 #태그5

**썸네일 아이디어**:
1. 구체적인 썸네일 아이디어
2. 구체적인 썸네일 아이디어
```

## 콘텐츠 작성 원칙

### 1. 분량
- 기본: 2,500~3,500자
- 애드센스 최적화: 3,000자 이상 권장
- 사용자 지정 시 해당 분량 준수

### 2. SEO 최적화
- 메인 키워드: 제목에 1회, 본문에 자연스럽게 3-5회
- 관련 키워드: 섹션 제목과 본문에 분산
- 키워드 밀도: 2-3% 권장
- **과도한 키워드 나열 금지**: 자연스러운 문맥 필수

### 3. 제목 사용 규칙 (## 중심 구조)
- `#` 메인 제목: 1회만, 핵심 키워드 포함
- `##` 주요 섹션: 3-5개, 관련 키워드 자연스럽게
- `###` 소제목: **가급적 지양**, 꼭 필요한 경우에만 최소한으로 사용
- ## 중심의 명확한 구조로 SEO 효과 극대화

### 4. 심층 작성 (충실한 분량)
- 각 섹션당 **반드시** 최소 6-8문장 이상 작성
- 피상적인 설명 배제, 심층적인 내용으로 구성
- 구체적인 수치, 통계, 데이터 필수 포함
- 검증된 사례와 실용적인 팁 제공
- 표를 활용한 비교 분석
- 리스트로 명확한 정보 정리
- 독자에게 실질적 가치 제공

### 5. 팩트 기반 작성 (허위 경험 절대 금지)
- 객관적 정보와 검증된 사실만 활용
- **절대 금지**: "제가 직접 써보니", "어제 다녀온", "직접 사용해보니", "제가 먹어본 결과" 등 가짜 체험담
- 허위 사용 후기 및 경험담 작성 절대 금지
- 출처가 명확한 데이터와 통계만 활용
- 데이터 인용 시 시기 반드시 명시
- 구체적인 수치와 실제 사례로 신뢰도 제고

### 6. 후킹 도입부
- 흥미로운 수치나 통계로 시작
- 독자의 문제나 궁금증 직접 언급
- 글을 읽어야 할 명확한 이유 제시
- 검색 의도 파악 후 즉시 답변 제시

### 7. 가독성
- 한 문단: 3-5문장 구성
- 마크다운으로 시각적 구조화
- 표와 리스트 적극 활용
- 적절한 줄바꿈과 여백
- 전문 용어는 쉬운 설명 병기

### 8. 필수 포함 요소
- **핵심 요약(마무리)**: 검색자의 질문에 대한 명확한 답변 포함
- **썸네일 아이디어**: 클릭을 유도하는 구체적 아이디어 2개
- **태그**: # 형식으로 5-10개

- [ ] 표나 리스트를 활용했는가?
- [ ] 각 섹션이 6-8문장 이상인가?
- [ ] 가짜 체험담이나 허위 정보가 없는가?
- [ ] 후킹 도입부가 검색 의도를 반영하는가?
- [ ] 문단별 3-5문장 구성이 지켜졌는가?
- [ ] 핵심 요약이 검색자의 질문에 답하는가?
- [ ] 썸네일 아이디어가 클릭 유도적인가?
- [ ] 태그가 # 형식으로 작성되었는가?
- [ ] 분량이 3,000자 이상인가?

## 애드센스 특화 주의사항

### 1. 콘텐츠 품질 (Google 정책 준수)
- **독창성**: 복사/붙여넣기 금지, 독자적 관점 필수
- **정확성**: 검증된 정보만 제공
- **전문성**: 깊이 있는 분석과 설명
- **신뢰성**: 출처 명확, 과장 금지

### 2. 금지 사항
- 허위/과장 정보
- 가짜 경험담이나 사용 후기 (실제 경험하지 않은 내용)
- 클릭베이트성 제목 (내용과 불일치)
- 키워드 스터핑 (부자연스러운 키워드 나열)
- 의미 없는 문장 반복 (늘어쓰기 금지)
- 얘은 콘텐츠 (가치 없는 내용)
- 저작권 침해

### 3. 권장 사항
- 실질적 가치 제공
- 독자 문제 해결 중심
- 최신 정보 업데이트
- 명확한 구조와 논리
- 읽기 쉬운 문체

### 4. SEO vs 독자 균형
- SEO는 수단, 독자 만족이 목적
- 키워드는 자연스럽게, 문맥에 맞게
- 검색 의도 파악 후 충실한 답변
- 가독성을 해치지 않는 선에서 최적화

## 분량 가이드

- **최소**: 2,500자
- **권장**: 3,000~3,500자
- **심화 주제**: 4,000자 이상 가능
- **중요**: 분량보다 품질, 늘어쓰기 금지
